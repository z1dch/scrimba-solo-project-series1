# 🚀 Scrimba Solo Project Series 1

This is the first solo project from the Scrimba series, built entirely from scratch using only **HTML and CSS**. 

The main goal here was to get comfortable with basic syntax and move away from just watching tutorials to actually writing code myself.

## Why I Built This

I wanted a solid playground to practice positioning and layouts. Instead of spending hours trying to design something from scratch, I used an AI tool to generate a web design mockup and some placeholder text. My job was simply to look at the design and figure out how to build it using code.

### What I Focused On:
*   **Flexbox & CSS Grid:** Getting a real feel for when to use Flexbox versus when Grid makes more sense.
*   **Clean HTML Structure:** Making sure my markup is easy to read and logically organized.
*   **Vanilla CSS:** No frameworks, no Bootstrap, and no shortcuts just pure CSS.

> 📝 **Note:** Since the design and text are AI-generated, the website's content doesn't represent a real business. It’s strictly a project to showcase my frontend fundamentals.
